# 053-Tcl

## Purpose
The Tcl package contains the Tool Command Language.

## Prerequisites
- You must be inside the `chroot` environment.
- Sources for Tcl-8.6.17 must be available.

## Step-by-Step Procedure
1. Extract the sources:
   ```bash
   tar -xf tcl8.6.17-src.tar.gz
   cd tcl8.6.17
   ```
2. Enter the unix directory and configure:
   ```bash
   cd unix
   ./configure --prefix=/usr           \
               --mandir=/usr/share/man \
               --disable-static
   ```
3. Build and install:
   ```bash
   make
   sed -e "s@^\(TCL_SRC_DIR='\).*@\1/usr/include'@" \
       -e "s@^\(TCL_BUILD_STUB_LIB_SPEC='\).*@\1-L/usr/lib @\2-ltclstub8.6'@" \
       -e "s@^\(TCL_BUILD_STUB_LIB_PATH='\).*@\1/usr/lib/libtclstub8.6.a'@" \
       -i tclConfig.sh
   make install
   ```
4. Install the Tcl headers:
   ```bash
   make install-private-headers
   ln -v -sf tclsh8.6 /usr/bin/tclsh
   chmod -v 755 /usr/lib/libtcl8.6.so
   ```

## Expected Outcome
The Tcl shell and libraries are installed.
