# 107-Vim

## Purpose
The Vim package contains a powerful text editor.

## Prerequisites
- Ncurses must be installed.
- Sources for Vim-9.2.0078 must be available.

## Step-by-Step Procedure
1. Extract the sources:
   ```bash
   tar -xf vim-9.2.0078.tar.gz
   cd vim-9.2.0078
   ```
2. Define the location of the configuration file:
   ```bash
   echo '#define SYS_VIMRC_FILE "/etc/vimrc"' >> src/feature.h
   ```
3. Configure for the system:
   ```bash
   ./configure --prefix=/usr
   ```
4. Build and install:
   ```bash
   make
   make install
   ```
5. Create a symlink for compatibility:
   ```bash
   ln -sv vim /usr/bin/vi
   for L in  /usr/share/man/{usr,ca,de,fr,it,ja,pl,ru,tr}/man1/vim.1; do
       ln -sv vim.1 $(dirname $L)/vi.1
   done
   ```
6. Create a basic configuration file:
   ```bash
   cat > /etc/vimrc << "EOF"
   " Begin /etc/vimrc

   set nocompatible
   set backspace=2
   set mouse=
   syntax on
   if (&term == "xterm") || (&term == "putty")
     set background=dark
   endif

   " End /etc/vimrc
   EOF
   ```

## Expected Outcome
The Vim editor is installed and configured with a basic `/etc/vimrc`.
