# 030-Sed

## Purpose
The Sed package contains a stream editor used to perform basic text transformations on an input stream.

## Prerequisites
- The cross-toolchain must be installed.
- Sources for Sed-4.9 must be available.

## Step-by-Step Procedure
1. Extract the sources:
   ```bash
   tar -xf sed-4.9.tar.xz
   cd sed-4.9
   ```
2. Configure for cross-compilation:
   ```bash
   ./configure --prefix=/usr --host=$SOS_TARGET --build=$(./build-aux/config.guess)
   ```
3. Build and install to `$SOS`:
   ```bash
   make
   make DESTDIR=$SOS install
   ```

## Expected Outcome
The `sed` binary is installed in `$SOS/usr/bin`.
