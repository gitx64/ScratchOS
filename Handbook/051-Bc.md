# 051-Bc

## Purpose
The Bc package contains an arbitrary precision numeric processing language.

## Prerequisites
- You must be inside the `chroot` environment.
- Sources for Bc-7.0.3 must be available.

## Step-by-Step Procedure
1. Extract the sources:
   ```bash
   tar -xf bc-7.0.3.tar.xz
   cd bc-7.0.3
   ```
2. Configure for the system:
   ```bash
   CC=gcc ./configure --prefix=/usr -G -O3 -y
   ```
3. Build and install:
   ```bash
   make
   make install
   ```

## Expected Outcome
The `bc` and `dc` utilities are installed in `/usr/bin`.
