# 045-Bzip2

## Purpose
The Bzip2 package contains programs for compressing and decompressing files.

## Prerequisites
- You must be inside the `chroot` environment.
- Sources for Bzip2-1.0.8 must be available.

## Step-by-Step Procedure
1. Extract the sources:
   ```bash
   tar -xf bzip2-1.0.8.tar.gz
   cd bzip2-1.0.8
   ```
2. Apply patches if needed:
   ```bash
   patch -Np1 -i ../bzip2-1.0.8-install_docs-1.patch
   ```
3. Prepare for compilation:
   ```bash
   sed -i 's@\(ln -s -f \)$(PREFIX)/bin/@\1@' Makefile
   sed -i "s@(PREFIX)/man@(PREFIX)/share/man@g" Makefile
   ```
4. Build and install:
   ```bash
   make -f Makefile-libbz2_so
   make clean
   make
   make PREFIX=/usr install
   ```
5. Install the shared library:
   ```bash
   cp -av libbz2.so.* /usr/lib
   ln -sv libbz2.so.1.0.8 /usr/lib/libbz2.so
   cp -v bzip2-shared /usr/bin/bzip2
   ```

## Expected Outcome
The Bzip2 utilities and libraries are installed.
