# 101-Kbd

## Purpose
The Kbd package contains key-table files, console fonts, and keyboard utilities.

## Prerequisites
- You must be inside the `chroot` environment.
- Sources for Kbd-2.7.1 must be available.

## Step-by-Step Procedure
1. Extract the sources:
   ```bash
   tar -xf kbd-2.7.1.tar.xz
   cd kbd-2.7.1
   ```
2. Apply a patch to fix issues with the check-syntax script:
   ```bash
   patch -Np1 -i ../kbd-2.7.1-backspace-1.patch
   ```
3. Configure for the system:
   ```bash
   sed -i 's/\(RETR_VAL=\)1/\10/' tests/libkeymap-test.sh
   ./configure --prefix=/usr --disable-vlock
   ```
4. Build and install:
   ```bash
   make
   make install
   ```
5. Install the documentation:
   ```bash
   mkdir -pv /usr/share/doc/kbd-2.7.1
   cp -R -v docs/man /usr/share/doc/kbd-2.7.1
   ```

## Expected Outcome
The keyboard and console utilities are installed.
