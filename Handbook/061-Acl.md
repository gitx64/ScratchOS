# 061-Acl

## Purpose
The Acl package contains utilities for administering Access Control Lists, which are used to define more fine-grained discretionary access rights for files and directories.

## Prerequisites
- Attr must be installed.
- Sources for Acl-2.3.2 must be available.

## Step-by-Step Procedure
1. Extract the sources:
   ```bash
   tar -xf acl-2.3.2.tar.gz
   cd acl-2.3.2
   ```
2. Configure for the system:
   ```bash
   ./configure --prefix=/usr         \
               --disable-static      \
               --docdir=/usr/share/doc/acl-2.3.2
   ```
3. Build and install:
   ```bash
   make
   make install
   ```

## Expected Outcome
The `getfacl` and `setfacl` utilities and libraries are installed.
