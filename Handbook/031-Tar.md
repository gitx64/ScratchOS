# 031-Tar

## Purpose
The Tar package provides the ability to create archives and manipulate existing archives.

## Prerequisites
- The cross-toolchain must be installed.
- Sources for Tar-1.35 must be available.

## Step-by-Step Procedure
1. Extract the sources:
   ```bash
   tar -xf tar-1.35.tar.xz
   cd tar-1.35
   ```
2. Configure for cross-compilation:
   ```bash
   ./configure --prefix=/usr                     \
               --host=$SOS_TARGET                \
               --build=$(build-aux/config.guess)
   ```
3. Build and install to `$SOS`:
   ```bash
   make
   make DESTDIR=$SOS install
   ```

## Expected Outcome
The `tar` binary is installed in `$SOS/usr/bin`.
