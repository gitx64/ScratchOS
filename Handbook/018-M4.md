# 018-M4

## Purpose
The M4 package contains a macro processor. It is used as a tool by many other packages during their build processes.

## Prerequisites
- The cross-toolchain must be installed.
- Sources for M4-1.4.21 must be available.

## Step-by-Step Procedure
1. Extract the sources:
   ```bash
   tar -xf m4-1.4.21.tar.xz
   cd m4-1.4.21
   ```
2. Configure for cross-compilation:
   ```bash
   ./configure --prefix=/usr   \
               --host=$SOS_TARGET \
               --build=$(build-aux/config.guess)
   ```
3. Build and install to `$SOS`:
   ```bash
   make
   make DESTDIR=$SOS install
   ```

## Expected Outcome
The `m4` binary is installed in `$SOS/usr/bin`.
