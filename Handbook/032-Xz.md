# 032-Xz

## Purpose
The Xz package contains programs for compressing and decompressing files, providing high compression ratios.

## Prerequisites
- The cross-toolchain must be installed.
- Sources for Xz-5.8.2 must be available.

## Step-by-Step Procedure
1. Extract the sources:
   ```bash
   tar -xf xz-5.8.2.tar.xz
   cd xz-5.8.2
   ```
2. Configure for cross-compilation:
   ```bash
   ./configure --prefix=/usr                     \
               --host=$SOS_TARGET                \
               --build=$(build-aux/config.guess) \
               --disable-static                  \
               --docdir=/usr/share/doc/xz-5.8.2
   ```
3. Build and install to `$SOS`:
   ```bash
   make
   make DESTDIR=$SOS install
   ```

## Expected Outcome
The `xz` utility and its related libraries are installed in `$SOS/usr/bin` and `$SOS/usr/lib`.
