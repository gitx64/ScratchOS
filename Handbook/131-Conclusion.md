# 131-Conclusion

## Purpose
Congratulations! You have successfully built ScratchOS 13.0 (systemd) from scratch.

## Prerequisites
- You must have followed all the steps in this handbook.

## Step-by-Step Procedure
Now that the system is built and unmounted, you can reboot your machine and select "ScratchOS" from the GRUB menu.

If everything went correctly, you will be greeted with a login prompt. You can log in as `root` (you will need to set a password during the first boot or in the chroot environment before finishing).

## Expected Outcome
A bootable, functional Linux distribution that you built yourself.
